# Nowplace

## 만든 목적
바로 지금! 위치를 기록하고 그 위치를 검색하고 싶은데, 기록할때 지도 앱들어가고 즐겨찾기에 등록하고 하는 과정이 번거로워서, [Naver map api]() 와 `HTML, CSS, Vanilla` 를 활용하여 만들어 봤어요! ✨
